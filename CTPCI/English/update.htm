Update

 

Chip Xilinx XC95288XL

March  2012 : CTPCI_1M  

 

- Ethernet board is running ! 

  Force to 0  the unused bits of the INT. registers IPEND & ENBI.

Fev  2012 : CTPCI_1L  

 

- The USB mouse is runnng ! 

  Correction for the BYTE SELECT [0:3] swapping.

  ATTENTION : USB 2 devices (using EHCI controller) cannot run because no driver in the 2.01 boot.

Jan  2012 : CTPCI_1K  

 

- IDE port is now running and the rate is 2.5 times faster than the Falcon port.

  By default the F030 IDE port is the PRIMARY and the new port is the SECONDARY.

  A button in the CPX BOOT menu allows to inverse the 2 IDE ports.

  WARNING, see the fitting documentation because there is an IRQ wire to connect between CTPCI & F030.

  It is not recommended to use the new port with too old Devices (before 1998) that may not support transfers with PIO MODE4.

Jan  2012 : CTPCI_1J    

           - Modification of the PCI Arbiter for PLX parking on PCI.

             The PCI cards get now the PCI bus correctly.

             NEC USB card now run with Radeon + SDMA/Blitter.

             Remove the RETRY protocol for 060 when PLX FIFO is full.

             Optimize TA# timing to PLX local master.

 

Dec  2010 : CTPCI_1F

           - new IEO (Int Enable for EtherNAT) timing for CTPCI + EtherNAT.

             NOT TESTED - EtherNAT must be updated too ! Contact NATURE.

 

June  2010 : CTPCI_1E

           - PCI zone extended to 2*1GB (no EtherNAT use).
           - TEAP output added for PLX DMA anti-freeze.

 

April  2010 : CTPCI_1C

           - Activation of the IEO output pin for EtherNAT. 

 

April  2010 : CTPCI_1B

           - Modification of the TA signal timing for better stability.

           - Change of I6 output and INT registers.

 

January 2010 : Initial version  - CTPCI_1A

 

 

 

To program the chip, you can contact me or buy a JTAG cable to transfer from // port to the chips.

This cable exists ready-to-use to many vendors. It will not run as it is on TOS machines (see below : 2 ).

Here are 2 links to buy (you can find many others) :

FRANCE : http://www.lextronic.fr/recherche.php?rech=jtag&numpage=10

CANADA : http://www.renaelectronics.com/product_x_cable.htm

 

 

Two ways :

 

 

1- Program with a PC

 

 

PC software is available free to www.xilinx.com. The name is IMPACT and you will find it in the WEBPACK to be downloaded. 

 

The good link to help you with PC, software & cable : http://nokturnal.pl/home/atari_16_32_bit/ct60_firmware_upgrade

You need too to reorganize the order of the wires on the JTAG  connector as on the schematics or pinout in the CT60 documentation (in the software pack). 

 

 

2- Program with the Falcon

 

 

To run the cable on your Falcon,  you need to modify it : disconnect the wire of pin #13 & connect to the pin #10 of the DB25 connector. You need too to reorganize the order of the wires on the JTAG  connector as on the schematics or pinout in the CT60 documentation (in the software pack). 

 

Other way : make the cable yourself.

Schematics of the cable for TOS (PDF) : Download ZIP file (15 KB). PC version schematics is on Xilinx web site.
