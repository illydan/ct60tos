OVERVIEW

RIORED-J is an ambitious project : a Power PC board designed to plug on the SLOT1 and later SOCKET370 of the PC mainboards.
RIORED-J brings a POWER PC with all IO of a PC motherboard, the few North Bridges compatible with bus 60x offering no such possibilities in a design of a new motherboard... 

An X86 emulation is executed from the FLASH in a SRAM and run the motherboard BIOS for hardware initialization. This emulation runs too the BIOS of the PCI daughter boards...

I stopped this project because my 'business angel' furnished me not enought financial ressources...

The content of RRJ project is not free. You need a written authorization if you want to use it.

HARDWARE

 

- PowerPC IBM G3cxe at 600 MHz.
- 4 MB FLASH for the BIOS and the X86 BIOS emulator.
- 512 KB SRAM for X86 emulation executed from the Flash.
- FPGA SPARTAN 2E XC2S150 for interface between the bus protocols X86 and 60x.
