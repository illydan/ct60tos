Material Overview :

The CT60 offers two modes : 030 and 060. The 030 mode (CT60 OFF) permits to keep a Falcon 100% as origin. With 060 mode (CT60 ON), it is too possible to boost the Falcon motherboard with some solders. 

Features (in MHz) table of the different settings

  CT60 OFF CT60 ON CT60 ON
FITTING SOLDERS & NO SOLDERS NO SOLDERS SOLDERS
CPU 030@16 060@66 060@66
FPU 882@16 or 32 060@66 060@66
BUS (F30/CT60) 16/OFF 16/66 25/66
DSP / VIDEL 32 32 50

The modes CT60 ON with solders at 50 MHz are available in VGA standard & extended résolutions. To use the RGB & ST-MONO (SM12x/144), a 32 MHz clock must be added on the VIDEL pin#14 input (EXT). This clock should be taken from the 32 MHz oscillator at left of  the DSP.

CPX

FILEINFO.CPX is an utility permitting to adjust 3 bits of program configurations and accessories: FastLoad, TT-Ram (Mem) and TT-Ram (Prg). 

The FastLoad bit prevents the system to erase all the memory to every launching of the software, that can take time with 64 MB of Fast-Ram.

The TT-Ram bit (PRG) loads the program in Fast-Ram, attention to the software audio and video (by example : CUBASE AUDIO will not output sounds if the PRG bit is set ! But you should set the MEM bit).

The TT-Ram bit (MEM) fact that the program will make this allowances memory in Fast-Ram, same remark for the program.
